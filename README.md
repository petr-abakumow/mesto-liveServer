# Проект 4: Место
--------------------------------------------------------------

## О проекте

Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

### Используемые технологии

1. HTML5
2. CSS3
3. java-script
4. Flex-box
5. Форма
6. popap
